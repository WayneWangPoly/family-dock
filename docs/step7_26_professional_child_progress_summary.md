# Step 7.26 Professional AI Child Progress Summary

## Why this matters

This is one of the highest-value parts of the app.

A normal calendar tells parents what happened.
A professional progress summary tells parents:

- what changed
- what is improving
- what is unstable
- what to practise
- what to ask the teacher/coach
- what evidence is missing

## Evidence model

The summary function reads:

```text
learning_notes
homework_tasks and homework_items
calendar_events
```

It stores source IDs for traceability.

## Report structure

Each report contains:

- executive summary
- narrative report
- strengths
- concerns
- observed patterns
- recommendations
- parent actions
- child actions
- teacher/coach questions
- next goals
- missing evidence
- confidence
- evidence count

## Professional safeguards

The AI prompt explicitly prevents:

- invented claims
- diagnosis
- vague motivational language
- shaming language
- unsupported teacher/coach opinions
