# Step 7.7 Manual Forms + Uploads

This package adds non-AI manual entry paths.

## Why

The AI assistant is the main comfortable entry, but parents still need direct controls:
- quick correction
- manual creation
- file upload
- request entry for children/homestay

## Storage note

The package creates a private storage bucket. Uploads save metadata to `homework_attachments`.
